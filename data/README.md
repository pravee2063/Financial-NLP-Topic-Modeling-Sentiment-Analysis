# Dataset

This project uses the **FinancialPhraseBank** dataset for topic modeling and sentiment analysis.

The dataset is **not included** in this repository due to its licensing terms.

## Download

You can obtain the dataset from:

- Kaggle: https://www.kaggle.com/datasets/ankurzing/sentiment-analysis-for-financial-news

After downloading, place the CSV file in the following location:

```text
data/all-data.csv
```

The notebook expects the dataset at this path.

## Dataset Citation

Malo, P., Sinha, A., Korhonen, P., Wallenius, J., & Takala, P. (2014). *Good Debt or Bad Debt: Detecting Semantic Orientations in Economic Texts*. Journal of the Association for Information Science and Technology.
